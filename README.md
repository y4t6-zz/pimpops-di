### Start project

1. Update composer:

```shall
composer update
```

2. Create table in MySQL named 'pimpops-di';

3. Import data from file 'pimpops-di.sql' into table 'pimpops-di';

4. Check connection config in file 'engine/Core/Database/Connection.php';
